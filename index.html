Assessment Master Builder（最终版）

支持 basic txt / pre json / post json / Excel 可一次上传一个人或多人 忽略 TXT 随机 id 按 clinical id + 日期为准 自动 pre/post 归并 简繁体姓名尽量自动合并 支持手工强制绑定日期 本地总库累计保存

① 上传文件 / 粘贴 TXT

说明：
• basic txt：支持粘贴，也支持上传 .txt/.tsv/.csv
• clinical：支持 .json / .xlsx / .xls / .csv
• 你以后可以每次只传一部分文件，点“追加到本地总库并预览”，它会自动累计

② MRI 顺序表（可编辑）

表里有的人按表排序；表里没有的人自动排到最后。

③ 别名归并规则（最稳）

程序会先自动做大小写/空格/标点/简繁体归一，再应用这里的 alias。

④ 强制绑定规则（可选）

用途：如果某个 ID 的自动匹配有歧义，可以手动指定其 pre/post 应使用的日期。
它会影响：
• 选择哪条 clinical 记录
• 给 pre/post 匹配哪条 txt
