BASIC INFO 批量整理器

可以連續追加上傳很多份 Excel / JSON / CSV。程式會把資料累積到同一張主表，遇到詳細評估 JSON 會自動計算 UPDRS 總分 和 TUG 平均秒數，最後統一導出一張 Excel。

輸入與追加

支援兩種資料來源：① 複製貼上的表格文字；② 單份或多份檔案。每次匯入都會「追加」進目前主表，而不是覆蓋。

批量上傳檔案

支援 .xlsx / .xls / .csv / .tsv / .txt / .json，可一次選多個檔案（例如 40 份）。

預設值與管理

沒有對應資料就留空；若有固定欄位可在此批量補上。資料會自動暫存到瀏覽器（localStorage），重新打開頁面還在。
