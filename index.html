1) basic TXT（tab 分隔）

贴整段 basic TXT（包含表头）；会忽略其中随机 id；同一人出现两次时，会按时间戳 + MRI 日期归到 pre / post。

2) 上传临床文件

支持： .json / .csv / .xlsx / .xls / .txt
普通横向表格会自动读取；像「編號 / 姓名 / 測試日期」这种中文表单型 CSV / Excel 也会自动识别。

3) 手工规则（可选）

输出规则：
1. 所有人都以 MRI master 的 Parti / Name / Date / Group 为准；
2. clinical 文件里的名字会尽量合并到 MRI 标准名（简繁体 + alias + 拼音/英文名映射）；
3. MRI 名单里没有的人，按识别顺序追加到总表末尾；
4. 导出 Excel 时第一行会自动做成分区表头：BASIC INFO / PRE Assessment / Post Assessment。
5. clinical 文件里的 So42 / S042 / T042 现在都会视为同一人（T042）。
