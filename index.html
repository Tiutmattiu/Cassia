BASIC INFO 批量整理器（固定編號 / 固定組別 / 固定 pre-post 時間）

這個版本會優先使用固定排程表決定每個人的 T 編號、Group、pre_date、post_date2。也就是說：不再信任來源表格上的 id / Group / pre-post 日期。

輸入與追加

可持續追加上傳很多份 Excel / JSON / CSV。系統會根據固定排程表把所有資料補進同一張 master 表。

批量上傳檔案

支援 .xlsx / .xls / .csv / .tsv / .txt / .json，可一次選多個檔案（例如 40 份）。

規則與管理

固定排程表優先級最高：ID / Group / pre-post 日期全部來自排程表。若來源檔有詳細評估 JSON，會自動算 UPDRS 總分 與 TUG 平均秒數。
