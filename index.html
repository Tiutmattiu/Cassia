Paste A + Review Many B → C Excel

A：直接粘贴 1–2 行（固定表头，不上传）
B：可以一次上传很多个；默认都是同一个人，但你可以在审核表里手动决定哪些文件要合并、哪些不要
重点：系统不会再因为 `張學強 / 张学强` 这种差异直接中止，而是让你在审核表里把它们改成同一个 canonical subject。

Step 1. 粘贴 A

使用方式：
直接从 Excel 复制 1–2 行数据 贴到下面。
不用贴表头；如果你把表头一起贴了，程序会尝试自动跳过。

规则：
- 贴 1 行 → 当作 pre
- 贴 2 行 → 优先按 timestamp 排序：早 = pre，晚 = post；若无法识别时间，则按粘贴顺序
A 只接受同一个人的 1–2 行，不会批量处理多人。

Step 2. 上传 B 并设置别名规则（可选）

这版不会立刻因为名字不一致报错。
你可以先全部上传，再在下面的 B 审核表 里决定：
1. 哪些文件参与计算（Include）
2. 哪些是 pre / post / unknown（Phase）
3. 哪些名字其实是同一个人（Canonical Subject）
这些规则只是帮你在初始审核表里自动填 canonical subject。
最终还是以审核表里每一行的 Canonical Subject 为准。

Step 3. B 文件审核表（关键）

你现在要做的事：
1. 看每一行是不是你要的文件
2. 不要的文件取消勾选 Include
3. 如果 auto phase 不对，就改 Phase
4. 如果不同写法其实是同一个人，就把 Canonical Subject 改成一样

例如：
- `張學強` → 改成 `张学强`
- `张学强` → 保持 `张学强`
这样它们就会被视为同一个人
