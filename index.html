MRI 总表合并与 Excel 导出

支持： basic TXT 复制粘贴 JSON / Excel 上传 按 MRI 名单强制归一 未识别姓名候选建议 手工绑定 pre/post

1) 贴上 basic TXT（tab 分隔）

会忽略 basic TXT 里的随机 id；同一人出现两次时，会优先按时间戳 + MRI 日期自动归到 pre/post。

2) 上传 pre/post JSON 或 Excel

可一次上传一个或多个文件；Excel 支持单表或多表。系统会优先使用 testType + id + testDate，并以 MRI 预设名单与日期为准。

3) 手工规则（可选）

强制绑定某个 ID 的 pre/post 日期
额外别名（可选，一行一个）

状态

输出规则：
1. 所有人都以 MRI 名单中的 Parti / Name / Date / Group 为准；
2. 临床文件里的名字会尽量合并到 MRI 标准名（简繁体 + 别名 + 拼音/英文名映射）；
3. MRI 名单里没有的人，按识别顺序追加到总表末尾；
4. 导出 Excel 时第一行会自动做成分区表头：BASIC INFO / PRE Assessment / Post Assessment。
